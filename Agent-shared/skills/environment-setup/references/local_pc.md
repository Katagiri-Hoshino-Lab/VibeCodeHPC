# Local PC Environment Setup

## Prerequisites
- tmux
- Python 3.10+
- One or more AI coding CLIs

## CLI Installation

### Claude Code
```bash
npm install -g @anthropic-ai/claude-code
```

### Codex
```bash
curl -fsSL https://github.com/openai/codex/releases/latest/download/install.sh | bash
```

### Gemini CLI
```bash
npm install -g @google/gemini-cli
```

### OpenCode
```bash
curl -fsSL https://opencode.ai/install | bash
```

## Authentication
Each CLI requires separate authentication — refer to CLI documentation.
